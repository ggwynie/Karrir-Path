Deskripsi
KarrirPath adalah platform karir berbasis web yang menggabungkan direktori lowongan kerja, sistem mentorship, dan generator roadmap karir berbasis AI dalam satu ekosistem yang terintegrasi. Platform ini dikembangkan sebagai respons atas tingginya angka mismatch antara kompetensi lulusan perguruan tinggi dengan kebutuhan industri di Indonesia.
Pengguna dapat menelusuri lowongan kerja, terhubung dengan mentor profesional, dan mendapatkan panduan karir yang dipersonalisasi menggunakan teknologi Google Gemini API — semuanya dalam satu platform.

Fitur
Autentikasi & Manajemen Peran

Login dengan email/password atau Google OAuth
Tiga peran pengguna: Pencari Kerja, Mentor, dan Recruiter
Routing otomatis ke dashboard sesuai peran setelah login

Lowongan Kerja

Penelusuran lowongan berdasarkan kategori: Full Time, Part Time, Freelance, dan Magang
Fitur filter dan pencarian

Mentorship

Pencari kerja dapat menelusuri mentor, memfilter berdasarkan keahlian dan ketersediaan, serta menjadwalkan sesi konsultasi
Chat 1-on-1 dengan AI yang mensimulasikan sesi mentoring
Dashboard mentor dengan tab Overview, Pesan, Jadwal, Ulasan, dan Analitik Pembayaran
Model bagi hasil: 80% untuk mentor, 20% untuk platform

Direktori Perusahaan

Profil perusahaan mitra lengkap dengan rating, loyalitas karyawan, dan informasi benefit

Roadmap Karir Berbasis AI

Pengguna mengisi latar belakang pendidikan dan target karir
Sistem menghasilkan roadmap karir yang dipersonalisasi menggunakan Google Gemini API
Hasil disimpan ke database dan ditampilkan dalam format Markdown

Profil Pengguna

Manajemen informasi personal, keahlian, pengalaman, pendidikan, CV, dan sertifikat
Indikator progres kelengkapan profil

Dashboard Recruiter

Pipeline kandidat: Baru → Ditinjau → Interview → Ditawari
Match score kandidat
Manajemen dan posting lowongan

Lisensi
Proyek ini dikembangkan untuk keperluan kompetisi akademik. Seluruh hak cipta dimiliki oleh tim pengembang.
